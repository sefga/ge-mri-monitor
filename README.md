# GE MRI Magnet & Cryo Monitor

Web-based tool to monitor GE MRI Cryocompressor and Coldhead parameters via IP scraping.

## Features
- Real-time parameter scraping from web interface.
- Automatic screenshot capture for manual verification.
- History of scans saved in JSON.

## Setup
1. `npm install`
2. `node server.js`
3. Open http://localhost:3000
